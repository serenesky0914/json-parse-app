# JSON Parse App

## Install

Install dependencies once:

```
npm install
```

## Run

To run the APP:

```
npm start
```

## Sample Json Data

src/App.js is sample json data line 8:

```
const sampleJsonData = {
  'array': [1, 2, 3],
  'boolean': true,
  'null': null,
  'number': 4,
  'object': {'a': 'b', 'c': 'd'},
  'string': 'Hello World'
};
```
You can init data like this
```
const sampleJsonData = {};
```

This will open a development server at http://localhost:3000
